Tablas HTML Y CSS

Tabla HTML

ETIQUETAS HTML DESCRIPCIÓN
body Para el contenido
head Se usa para información sobre el documento
div Usada para división dentro del documento
a Para enlaces
strong Se usa para poner el texto en negrita
br Usada para saltos de línea
h1...h6 Sirven para títulos dentro del contenido, siendo h1 para títulos grandes hasta llegar a h6 con títulos más pequeños
img Se usa para añadir imágenes al documento
ol Se usa para listas ordenadas
ul Usada para listas desordenadas
li Para elementos dentro de la lista
p Se usa para párrafos
span Con está etiqueta es posible personalizar el estilo de una parte del texto

Tabla CSS

ESTILOS CSS DESCRIPCIÓN SINTAXIS
Colors Con el lenguaje de CSS podemos cambiar el color de los elementos HTML. 
                        h1{
                         color:rgb(38, 99, 229);
                        }
                        p{
                         color:rgb(131, 36, 226);
                        }
Backgrounds Gracias al lenguaje de estilo de CSS podemos configurar el color de fondo de algunos elementos HTML. 
                        h1{
                         background-color: rgb(38, 99, 229);
                        }
Borders Con el lenguaje de CSS es posible configurar las propiedades de los bordes de algunos elementos HTML. 
                        h1{
                         border: groove;
                         border-color: rgb(0, 0, 129);
                        }
Margins Usada en CSS para configurar las propiedades de las márgenes para algunos elementos de HTML. 
                        div{
                         margin: 80px;
                         border: 1px solid #4CAF50;
                        }
Align Nos permite en el lenguaje CSS configurar propiedades de alineación para los elementos h1, h2, h3 y h4 en un documento HTML. 
                        h1{
                         text-align: center;
                        }
                        h2{
                         text-align: left;
                        }
                        h3{
                         text-align: right;
                        }
                        h4{
                         text-align: justify;
                        }
Fonts Usada en el lenguaje CSS para configurar las propiedades de fuente de algunos elementos HTML. 
                        h1{
                         font-family: sans-serif;
                        }
                        h2{
                         font-family: cursive;
                        }
Padding Usando el lenguaje de CSS podemos configurar las propiedades de relleno entre el contenido de un elemento HTML y sus bordes. 
                        div{
                         padding: 50px;
                         border: 1px solid rgb(0, 0, 129);
                        }
Width y Height Con el lenguaje de estilo de CSS logramos configurar las propiedades de ancho y alto para algunos elementos HTML. 
                        div{
                          width: 500px;
                          height: 100px;
                          background-color: rgb(0, 0, 129);
                        }
Link Con el lenguaje de estilo de CSS es posible configurar las propiedades de estilo de un link. 
                        /*Link no visitado*/
                        a:link{
                         color:red;
                        }
                        /*Link visitado*/
                        a:visited{
                        color:green;
                        }
                        /*Al pasar el mouse*/
                        a:hover{
                        color:yellow;
                        }
                        /*Link seleccionado*/
                        a:active{
                        color:blue;
                        }
List Usando el lenguaje de CSS podemos configurar propiedades de estilo de listas en HTML. 
                        ul.a{
                         list-style-type:square;
                        }
        
                        ol.b{
                         list-style-type:lower-alpha;
                        }
Tables Usando el lenguaje de CSS podemos configurar propiedades de estilo de tablas en HTML. 
                        table,td,th{
                         border:1px solid #ddd;
                         text-align:left;
                        }
                        th{
                         background-color:rgb(83, 117, 191);
                        }
                        td{
                         background-color:rgb(131, 164, 235);
                        }
                        table{
                         border-collapse:collapse;
                         width:100%;
                        }
                        th, td{
                         padding:15px;
                        }
Position Con este estilo es posible configurar las propiedades de posicionamiento de un elemento HTML, (Static, relative, fixed, absolute, sticky) 
                        div.relative{
                         position:relative;
                         width:500px;
                         height:230px;
                         border:3px solid #83A4EB;
                        }
        
                        div.absolute{
                         position:absolute;
                         top:80px;
                         right:0;
                         width:250px;
                         height:150px;
                         border:3px solid #83A4EB;
                        }
Overflow Nos permite configurar las propiedades de desbordamiento de un elemento HTML cuando su contenido es demasiado grande, (visible, hidden, scroll, auto) 
                        div{
                            background-color:#eee;
                            width:210px;
                            height:55px;
                            border:1px dotted black;
                            overflow:auto;
                        }
Float Nos sirve para configurar las propiedades de posicionamiento de un elemento HTML, (left, right, none, inherit) 
                        img{
                            float: right;
                        }
